# Анализ чата по ускорению 1С: Рекомендации по SSD (2025-2026)

На основе анализа сообщений чата (охватывающего период с начала 2025 по февраль 2026 года), был составлен список рекомендаций по выбору SSD для серверов и рабочих станций 1С.

## Мнение эксперта по брендам и сериям
Приоритетным источником рекомендаций являются мнения эксперта Вячеслава Гилёва (gilev.ru). Ниже приведены его обобщенные оценки по популярным брендам и сериям SSD для задач 1С.

### 1. Intel / Solidigm / Optane
*   **Общий вердикт:** **Эталон**.
*   **Intel Optane (P5800X):** Безусловный лидер. Используется как эталон надежности и производительности ("это не оптаны" — фраза, подчеркивающая, что другие диски хуже). Идеальны для логов транзакций и высоконагруженных баз.
*   **Intel P-серия (P4610 и др.):** Хорошие диски, но старые модели покупать рискованно ("их возраст ого-го", "не надо думать, что они вечные").
*   **Solidigm:** Рассматриваются как продолжатели Intel (D7 серия), отношение положительное.

### 2. Samsung (Enterprise)
*   **Общий вердикт:** **Смешанный (зависит от DWPD)**.
*   **PM1735 (3 DWPD):** Оценивается как **"годится"** для нагрузки среднего уровня (50-100 пользователей на диск).
*   **PM9A3 / PM1733 (1 DWPD):** Эксперт считает диски с ресурсом 1 DWPD **непригодными** для активных баз данных ("это плохо", "за гранью разумности"). Они подходят только для чтения, бэкапов или ОС.
*   **PM1763:** Новые модели (Gen5), по которым пока нет однозначной статистики, эксперт выразил сдержанность/неопределенность.

### 3. Kingston
*   **Общий вердикт:** **Четкое разделение по сегментам**.
*   **Серия DC (DC2000B, DC600M):** **Положительная** оценка для своих задач. DC2000B рекомендован как загрузочный диск для сервера (PLP, стабильность).
*   **Серия A400 (Consumer):** **Крайне негативная** оценка ("убогий дешманский QLC", "только по приговору суда"). Категорически нельзя использовать в серверах.

### 4. Micron (Enterprise)
*   **Общий вердикт:** **Скептический / Негативный**.
*   **Серия 9550 MAX:** Упоминается как более свежая и приемлемая альтернатива старым сериям.
*   **Серия 9300 PRO:** Версия PRO (1 DWPD) — **не годится**. Эксперт рекомендует смотреть только на серию **MAX** (с повышенным ресурсом).
*   **Серия 7450:** Прямая нерекомендация ("мы не рекомендуем", "выглядят плохо", "графики швах"). Отмечаются проблемы с производительностью и стабильностью.

### 5. Ресурс (DWPD) — Главный критерий
Гилёв формулирует жесткое правило:
*   **1 DWPD** (Read Intensive): **Не подходит** для баз данных 1С.
*   **3 DWPD** (Mixed Use): **Минимум** для рабочих баз (до 50-100 пользователей).
*   **Optane / Write Intensive**: Для высоких нагрузок.

## Ключевые тренды и наблюдения (2025-2026)

1.  **Важность драйверов:** Для **Samsung 980 PRO** установка проприетарного драйвера критична. Без него производительность в тестах 1С (типа Гилёва) может падать в 3-4 раза.
2.  **Приход Gen5:** В новых сборках конца 2025/2026 годов начинают использоваться диски PCIe Gen5 (Crucial T705, Biwin, Solidigm) для достижения максимальной скорости.
3.  **Анти-QLC:** Строгое предупреждение против дисков QLC (как Kingston A400, Micron 2400), так как они считаются ненадежными и медленными для бизнес-использования.
4.  **Актуальность Optane:** Несмотря на то, что технология Optane (P5800X) в некоторых контекстах считается "уходящей", она все еще цитируется как ультимативное решение для операций с базами данных, чувствительных к задержкам.

## Дополнительные заметки по выбору SSD
* **Ресурс записи (DWPD):** Для баз данных 1С критически важен показатель DWPD (Drive Writes Per Day). Эксперты настоятельно не рекомендуют использовать диски с 1 DWPD (как Samsung PM9A3) для активных баз данных, называя это "за гранью разумности". Такие диски подходят только для ОС, бэкапов или чтения.
* **NVMe vs SATA:** SATA SSD (даже Enterprise) становятся "узким местом". Для современных серверов 1С стандарт де-факто — NVMe.
* **Потребительские диски:** Использование Consumer SSD (даже PRO серий) в серверах допустимо только для очень маленьких баз или тестовых контуров. Проблема не только в ресурсе, но и в стабильности задержек под нагрузкой.
* **Драйверы:** Для Samsung NVMe (особенно Consumer серий 980/990 PRO) критически важно использовать проприетарный драйвер Samsung, а не стандартный Microsoft, это может дать прирост производительности в разы.
